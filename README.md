# My Website: [maxbossing.de](https://maxbossing.de)

This repository holds my [Website](https://maxbossing.de), built with [Hugo](https://gohugo.io)

## Development

1. Install Hugo
2. Clone this repository
3. To create a new post, execute `hugo new posts/<POST-NAME>.md` 
4. To preview changes, execute `hugo serve -D`

## Deployment

The website is automatically built by GitHub Actions, and deployed using GitHub pages
